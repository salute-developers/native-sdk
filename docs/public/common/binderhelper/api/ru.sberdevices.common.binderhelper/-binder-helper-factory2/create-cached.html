Создать закешированную версию BinderHelper. Используется для непродолжительных и частых IPC вызовов.

Parameters

context

Application context.

intent

Intent с компонентом сервиса, к которому будем подключаться.

loggerTag

тэг для логгира, если не передать, то будет создан логгер по умолчанию.

disconnectDelay

таймаут на отключение от удаленного сервиса. По умолчанию - 3 секунды.

getBinding

вызывается в коллбеке onServiceConnected() android.content.ServiceConnection. Дает биндеру интерфейс сервиса
